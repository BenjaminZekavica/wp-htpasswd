WP Htaccess improve the security of your WordPress site.



== Description ==

This plugin simplifies the life of WordPress users.
Now you can create easily a htpasswd file to improve the security.
Plugin available on [GitHub](https://github.com/MatteoCandura/wp-htpasswd).



== Installation ==

1. Upload `wp-htpasswd` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the "Plugins" menu in WordPress
3. Now you have a new admin page "WP htpasswd"
